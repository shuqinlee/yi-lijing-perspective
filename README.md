# yi-lijing-perspective

易立竞视角的思维顾问 Skill。

它不是“模仿一个尖锐的人说话”，而是把易立竞的采访哲学、提问方法和对话判断方式整理成一个可复用的思维框架：看见回避，拆开防御，追问前提，还原一个人真实的表达结构。

## 能做什么

- 分析一段对话里，谁在回避，回避了什么
- 把普通问题改成更能问出真话的问题
- 识别 PR 话术、情绪防御、转移话题和自我包装
- 用“自画像权利”“共舞与博弈”“缝隙就是入口”等模型分析人物
- 生成易立竞式追问清单，用于采访、谈判、复盘或困难沟通
- 分析易立竞本人采访风格，以及她和鲁豫、许知远、柴静等人的差异

## 适合什么时候用

当你想知道：

- 这个人真正想表达什么
- 这句话哪里不对劲
- 对方为什么一直绕开重点
- 怎么问才能让对方说真话
- 一个采访为什么让人不舒服
- 一个问题的前提是不是错的

就可以触发这个 Skill。

## 触发词示例

你可以这样说：

- “用易立竞的视角分析一下”
- “易立竞会怎么问”
- “帮我拆穿这个人在说什么”
- “怎么问出真话”
- “给我一组易立竞式追问”
- “这个问题的前提哪里不对”
- “分析这个访谈为什么让人不舒服”
- “易立竞和鲁豫有什么不同”

## 使用方式

### 1. 分析模式

用于理解人物、对话、采访场景。

示例：

```text
用易立竞的视角分析这段采访，看看被访者在回避什么。
```

### 2. 提问模式

用于生成问题清单或追问路径。

示例：

```text
我要采访一个创业者，帮我生成 10 个易立竞式追问。
```

### 3. 拆解模式

用于识别话术、防御和逻辑前提。

示例：

```text
帮我拆一下这段回应，他到底在保护什么？
```

### 4. 风格理解模式

用于分析易立竞本人的采访方法。

示例：

```text
易立竞为什么会让一些被采访者不舒服？
```

## 边界

这个 Skill 不是易立竞本人，只是基于公开信息整理出的视角模拟和分析工具。

不要用它来：

- 编造易立竞没有公开说过的私人信息
- 代表她对现实人物或事件发表“官方立场”
- 对活人做没有证据的道德判决
- 把尖锐提问变成攻击、审判或羞辱

易立竞式追问的核心不是“狠”，而是“准”。不是为了赢，是为了让真实浮出来。
